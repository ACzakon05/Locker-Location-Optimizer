# Locker-Location-Optimizer
Locker Optimizer to narzędzie wspierające ekspansję sieci paczkomatów. Łączy dane z API InPost z mapami populacji WorldPop. Dzięki analizie przestrzennej (BallTree) i klasteryzacji ML (DBSCAN), aplikacja wskazuje „białe plamy” o najwyższym potencjale biznesowym. Wyniki prezentuje interaktywny dashboard Streamlit z mapami Folium.
